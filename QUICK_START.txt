📋 STEP-BY-STEP GUIDE:

1️⃣  INSTALL DEPENDENCIES
    Open terminal in this folder and run:
    
    npm install
    
    ⏱️  Takes about 1-2 minutes

2️⃣  START DEVELOPMENT SERVER
    
    npm run dev
    
    🌐 Open browser at: http://localhost:3000

3️⃣  PERSONALIZE YOUR PORTFOLIO
    
    ✏️  Update Contact Info:
        File: components/Contact.tsx (lines 12-25)
        Change: Email, LinkedIn, GitHub links
    
    🖼️  Add Project Images:
        Folder: public/images/
        Add: coopayroll.png, cedarsreet.png, futureforward.png, 
             coopeng.png, peaceinitiative.png
    
    🔗 Update Project Links:
        File: data/projectsData.ts
        Change: "your-link-here.com" to real URLs

4️⃣  DEPLOY TO VERCEL (FREE)
    
    a) Push code to GitHub
    b) Go to vercel.com
    c) Click "New Project"
    d) Import your repository
    e) Click "Deploy"
    
    ✅ Live in ~60 seconds!

❓ TROUBLESHOOTING:

    Port 3000 in use?
    → Run: npx kill-port 3000

    Build errors?
    → Delete .next folder and node_modules
    → Run: npm install
    → Run: npm run build

    Images not showing?
    → Make sure files are in public/images/
    → Check file names match exactly (case-sensitive)
